# EasyStock
## 프로젝트 기획 배경
토스증권과 카카오증권은 복잡한 투자 정보를 누구나 이해할 수 있도록 설계된 쉬운 UI/UX를 통해 많은 사용자를 확보하고 있다.
그러나 여전히 초보 투자자에게는 “어떤 종목이 좋은지”, “실제 수익을 내는 투자자는 어떻게 포트폴리오를 구성하는지”에 대한 정보가 부족하다.

본 프로젝트는 특정 전문가(투자 경험자)의 종목 선택 기준과 포트폴리오를 정보 제공 형태로 시각화하여 보여주는 것을 목표로 한다.
투자 자문이나 대리 운용이 아닌, 공개된 투자 판단과 포트폴리오 구조를 참고할 수 있는 정보 서비스로 설계하여 법적·규제적 리스크를 최소화하였다.

또한 무료 사용자와 유료 구독자를 구분하여, 제공 정보의 깊이와 범위를 차등화함으로써 서비스 확장성과 수익 모델을 동시에 고려하였다. 이렇게 되어있는데 커밋메시지 추천 좀
